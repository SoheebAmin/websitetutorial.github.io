About This Website

Some ways this website can help you with HTML, CSS, and JavaScript



This website consists of 3 pages, not counting this one. Each page appears simple, but has a lot going on in the back end.

    The simplest page is the homepage, which uses a short video of waves using HTML5's video tag as its background
    The backgrounds page helps in understanding how different CSS stylesheets and JavaScript pages can effect the visual and interactive style of a webpage. The page adds a layer of compexity by allowing for the switching of stylesheets dynamically, as well as mixing between different CSS and JavaScript implementations. This is made by possible with JavaScript itself.
    The JavaScript page shows other uses for JavaScript besides background visuals. This includes running simple programs like a text scrambler, editing the HTML of the page via the page itself, and other miscellaneous features the language can bring to the interactivity of a website.

Let's Explain This Modal

Since this page is about explanations, let me take a second to explain this helpful alternative to the simple alert function.


This drop down box is called a modal. It is built using CSS and JavaScript. For this website's implementation, Bootstrap 4's pre-built modal is used, which also uses Ajax and JQuery for its implementation. In the source code, you can see the CSS and JavaScript pages imported to make this modal work. Then, at the bottom of the body of the HTML page, the code for modal itself is present.
